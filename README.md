# Secure-Key

SecureKey is a full-stack locksmith web app for a Witbank-based business. It lets customers browse services, request quotes, and shop online, while admins manage users, orders, and inventory via a secure dashboard. Built with Spring Boot, React, and MySQL.
📌 Overview
SecureKey is a web-based system built to improve customer experience and streamline locksmith business management. It replaces manual processes with a digital solution for enquiries, product purchases, and service requests.

🚀 Features
👤 Customer Side

View services and pricing

Browse products and add to cart
Request quotes and send enquiries
Secure checkout and order tracking

🛠 Admin Side
